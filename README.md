# 五强溪水电站 · 水利计算课程设计 在线计算系统

> 河海大学水文水资源学院《水利计算》课程设计配套计算工具
> 单文件网页 · 打开即用 · 无需安装任何环境

一个浏览器里就能跑的水利计算工具，覆盖五强溪水电站课程设计的**全部计算**：兴利计算、防洪计算、经济比较。原始数据均取自课设"原始数据及说明"，计算方法严格按《水利计算课程设计任务指示书》。给下一届做这个课设的同学用来**理解流程、实时调参、复核结果**。

## 功能

| 模块 | 内容 |
|---|---|
| **① 兴利计算** | 新径流系列（扣航运10 + 灌溉35 m³/s）→ 死水位三因素 → 保证出力 N_p（长系列**等出力法**、逐年变供水期、P=87.5%）→ 装机容量 → 水库调度图（防破坏线 / 加大出力辅助线 / 汛限水位）→ 重复容量、多年平均电能、径流利用系数 |
| **② 防洪计算** | 20 / 1000 / 10000 年一遇入库洪水调洪演算（Δt=5h）→ 防洪高水位 / 设计洪水位 / 校核洪水位、最大下泄量、总库容 → 风浪高 + 坝顶高程 |
| **③ 经济计算** | 投资、年运行费、替代火电站 + 配套煤矿、防洪效益 → 年费用最小法比选四方案 → 推荐最优正常蓄水位 |
| **成果汇总** | 自动生成四方案"水利指标总表"（即任务书表1） |

- 四个备选正常蓄水位方案（▽120 / 115 / 108 / 100）随时切换
- 所有关键参数（出力系数 K、设计保证率、安全泄量、折现率等）**可在线编辑、实时重算**
- 全部图表（Z-V / Z-q 曲线、洪水过程线、频率曲线、调度图、调洪演算过程线、年费用比较）均为网页内绘制，无外部依赖

## 计算方法说明

- **径流**：1950年4月起的31年（水文年4月—次年3月）逐月平均流量
- **保证出力**：长系列等出力操作，逐年取"低于该年年平均流量的最长连续枯水月段"为供水期，按 P=87.5%（经验频率 m/(n+1)）取设计枯水年
- **调洪**：溢洪坝自由堰流 Q=1.77nBH^1.5、中孔孔口出流 Q=μnω√(2gH)，起调水位取汛限水位
- **经济**：年费用最小法，折现率 r₀=0.1，水电 50 年、替代火电 25 年等额重置

> 计算口径、参数取值可在网页参数栏调整。结果供学习与复核参考，最终设计请结合规范与指导老师意见。

## 数据来源

- `DATA.DAT` / 径流系列表 — 坝址断面历年月平均流量
- `Z_v.dat` — 水位-库容曲线（库容单位 m³/s·月）
- `z_q.dat` — 下游水位-流量关系
- `设计洪水.xls` — 20 / 1000 / 10000 年一遇入库设计洪水过程线
- 泄洪建筑尺寸、经济资料 — 任务指示书表7~表11

## 部署说明（如何让别人也能在线打开）

1. 在 GitHub 新建一个**公开（Public）**仓库，名字建议 `wuqiangxi-hydro-calculate`
2. 把本仓库内容上传上去（含 `index.html`）
3. 进入仓库 **Settings → Pages**，Source 选 `Deploy from a branch`，Branch 选 `main` / 根目录 `/`，保存
4. 等一两分钟，访问 `https://你的用户名.github.io/wuqiangxi-hydro-calculate/` 即可
